## Notes

### Looping sounds (#7, #11, #30)
These are labelled *(loop)* above. The source recordings are single short takes — the game engine (Web Audio API `loop: true`) handles playback looping. If glitches appear at the loop point, trim a few ms from the tail with ffmpeg before implementation.

### Cue gaps
The following cues from the master list were **not sourced** in this batch — they are living-character sounds reserved for separate sourcing per task instructions:
- #2 Double jump, #3 Bounce, #23–26 (wall jump, float, fall, bounce land), #29 Boss appear, #31 Boss idle, #34 Boss aim, #37–38 (boss roar, stagger), #40–45 (creature sounds), #55 (powerup), #57+ (miscellaneous living sounds)

### Source packs
All Kenney packs are in `../audio/_sources/extracted/` (not committed). Re-download ZIPs at any time from kenney.nl under CC0.
OpenGameArt.org sounds are sourced directly from `opengameart.org/sites/default/files/`.
